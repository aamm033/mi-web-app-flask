Mi Trayectoria en Python y Flask

Esta aplicación fue construida como un proyecto de aprendizaje y refactorización. Inicié con una estructura simple y la transformé en una aplicación robusta utilizando las mejores prácticas de Flask, incluyendo:

  • El **Patrón de Fábrica** para la inicialización.
  • **Blueprints** para la modularidad (`main`, `auth`).
  • **Flask-Login** para la autenticación y **Autorización** (Roles de Administrador).
  • **Flask-Migrate** para la gestión de la base de datos.
  • **Flask-Bootstrap4** para una interfaz moderna y *responsive*.
